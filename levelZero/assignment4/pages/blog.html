5 Shorthand Javascript Tricks To Save Your Time

Shorthand Techniques are special kind of code writing techniques which helps you to reduce the code length and convert the code presentation into more clean and optimized looking way. So here are few examples you can write your Javascript codes into more clean way.

1) The Ternary operator

Instead of writing if else, which uses generally 3-4 lines we can use Ternary oprator to reduce the length. 


//Longhand 
let marks = 26; 
let result; 
if(marks >= 30){
  result = 'Pass'; 
} else { 
  result = 'Fail'; 
} 

//Shorthand 
let result = marks >= 30 ? 'Pass' : 'Fail';

2) Arrow Function

Arrow function is new addition to js in ES6, where user can create function in more clean way. 


//Longhand 
function add(num1, num2) { 
    return num1 + num2; 
} 

//Shorthand 
const add = (num1, num2) => num1 + num2;

3) Checking conditions

Instead of comparing element to boolean, we can directly put the element inside the if condition. 


//Longhand 
let val = true; 

if(val == true){
  console.log('Condition passed')
} else { 
  console.log('Condition Failed')
} 

//Shorthand 
let val = true; 

if(val){
  console.log('Condition passed')
} else { 
  console.log('Condition Failed')
}

4) Map function

Instead for traditional loop technique, we can use map function to traverse through each element of Array. 


//Longhand 
var str = [1, 2, 3, 4, 5 , 6];

for(let i = 0; i < str.length; i++) {
  console.log(str[i] * 2)
}

//Shorthand 
str.map(i => console.log(i * 2) )

5) Template Literals

We normally use + operator to concatenate string values with variables. With ES6 template literals we can do it in a more simple way. 


//Longhand 
console.log('You got a missed call from ' + number + ' at ' + time); 

//Shorthand 
console.log(`You got a missed call from ${number} at ${time}`);
